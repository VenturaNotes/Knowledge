- Information
	- Go is a "fast, statically typed, compiled language"
	- Go is a general purpose language
	- Go has built-in testing support
	- Go is an object-oriented language (in its own way)
- Can only have 1 main function
- Setting up Go
	- golang.org/dl
	- type `go` in terminal to make sure it's been installed correctly
	- Download `go` package in VS Code
	- Install all when suggested
