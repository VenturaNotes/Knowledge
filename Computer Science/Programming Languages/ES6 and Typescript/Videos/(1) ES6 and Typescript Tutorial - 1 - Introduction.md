- ES5
	- Javascript
		- What we call Javascript is one of the implementations of a standard specification called ECMA Script 
		- Javascript used when talking about language in practice
	- ECMA Script
	- Version 5
		- We use ES5 when referring to a standard
- ES2015
	- ECMA Script 2015
	- ES2015 aka ES6
		- Evolution of ES5
	- New Features
		- Classes, constants, Arrow functions
			- Arrow functions allow us to write shorter function syntax: [^1]
				- `let myFunction = (a,b) => a * b;`
	- ES2015 = ES5 + Awesome features
- Typescript
	- Free and Open source
	- Developed + maintained by Microsoft
	- Typed superset of JS
	- Compiles to plain javascript
	- Features of ES2015, ES2016
	- Types (Optional)
		- Numbers, strings and booleans
	- Safer code
		- Main benefit of typescript code
		- More bug free code
