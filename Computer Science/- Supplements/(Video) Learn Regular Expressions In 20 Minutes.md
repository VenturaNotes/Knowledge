## Info
- A regular expression is a way to search through a string of text
- Regular expression allows us to search through text in order to do things such as validation, or get certain pieces of the text, advanced find and replace, there is a ton of things we can do. Allows us to search through text in a very advanced way and group searches together to do powerful things such as find and replace with it.
- /cat/g
	- / (the first one)
		- Indicates the start of a regular expression
	- cat
		- matches these character in order and also case sensitive
	- / (the second one)
		- Indicates the end of a regular expression and the start of expression flags
	- g
		- "Global search" - Retain the index of the last match
