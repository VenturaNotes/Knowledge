## Statements, Properties and Quantifiers
- S(x): "Statement about x is true"
- P(x): "x has property P."
- Universal Quantifier
	- $\forall$ x $\in$ X: S(x)
		- "For every x $\in$ X, S(x) true"
		- $\forall$ is universal
- Existential Quantifier
	- $\exists$ x $\in$ X: S(x)
		- There exists an x $\in$ X s. t. S(x) is a true statement.
	- $\exists$! x $\in$ X : S(x)
		- There exists a unique element of capital X for which the statement is true
		- If it was P(x) instead of S(x)
			- There exists a unique element X that has property P
