	- B $\subset$ A
		- B is a subset of A
		- A is a superset of B
		- B is a [[proper subset]] of A
			- A has 1 or more elements different from the elements in B
	- C $\subseteq$ A
		- C is an [[improper subset]] of A
			- Elements present in set C are also present in set A
			- Both sets are equal (although arrangement of numbers can be different)
