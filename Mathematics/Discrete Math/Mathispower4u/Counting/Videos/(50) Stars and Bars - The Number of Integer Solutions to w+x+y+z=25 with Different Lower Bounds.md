	- Stars and Bars
		- How many integer solutions to $w + x + y + z = 25$ are there if $w \ge 1, x \ge 2, y \ge 3, \text { and } z \ge 4$?
	- More general case
		- If $w \ge 0, x \ge 0, y \ge 0, \text { and } z \ge 0$
		- Sum of 25 $\to$ 25 $*'s$
		- 4 distinguishable variables $\to$ ||| (3 bars)
	- Our specific case
		- Sum of 15 $\to$ 15 $*'s$
			- This happens if assigning the stars 1, 2, 3, and 4
		- 4 distinguishable variables $\to$ ||| (3 bars)
		- Number of integer solutions: $w + x + y + z = 15 \text { if } w \ge 0, x \ge 0, y \ge 0, \text { and } z \ge 0$
