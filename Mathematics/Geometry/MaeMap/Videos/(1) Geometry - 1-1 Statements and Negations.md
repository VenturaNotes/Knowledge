	- Statement - A sentence that is either true or false, not both
	- Truth value - true or false
	- I live in the U.S.A (p)
		- True, the YouTuber does live in the U.S.A
	- Today is Jan. 1st, 2015 (q)
		- False, the YouTuber created the video on a different date
	- Most common variables to represent statements are "p" and "q"
		- Truth value of p is true and truth value of q is false
	- Negation - Statement is given the opposite truth value
		- Symbol is "~"
