- [[Trails]]
	- <mark style="background: #FFF3A3A6;">No repeated edges</mark>
	- E - A - B - C - E - D
		- Open walk
	- Can be opened or closed
	- Trails are walks
	- Open trails
		- Start Finish different vertex
	- Closed Trail
		- Start Finish Same vertex
		- E - A - B - C - E
- [[Paths]] (think of posties) ^fc09d6
	- <mark style="background: #FFF3A3A6;">No repeated vertices (except first & last)</mark>
	- Edges can be the same
	- Example
		- C - B - A - E - D
	- [[Open Path]]
		- Start Finish Different vertex
	- [[Closed Path]]
		- Start Finish Same Vertex
- Example
	- C - D - D - E - C - B
		- Open trail 
			- No repeated edges
			- Can have repeating vertices
