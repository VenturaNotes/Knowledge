- Universal set (U)
	- A set that contains all elements under consideration
- ![[Screenshot 2023-01-10 at 12.10.43 PM.png]]
	- Complement of A is the set of all elements that are not in A
		- U - A = A'
	- Complement of the empty set is the universal set
		- $\varnothing'$ = U 
	- Complement of the universal set is the empty set
		- U' = $\varnothing$ 
- The complement of any generic set is the set of all elements not in the generic set.
