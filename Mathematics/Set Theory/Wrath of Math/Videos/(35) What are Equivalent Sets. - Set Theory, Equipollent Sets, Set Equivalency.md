	- [[Equivalent Sets]]
		- Two sets are equivalent if they have the same cardinality
		- Can write equivalent like $\equiv$ or $\sim$ 
	- If two sets are equal, then they are equivalent
		- converse not true
	- Two sets are [[equipollent]] if they have the same cardinality
		- Equipollent is less common so it might be more practical to use the word equivalent
