	- [[Union]]: $A \cup B = \{x | x \in A \text{ or } x \in B\}$
		- Definition in set-builder notation
		- Set containing all elements in either set
	- [[Intersection]]
		- Set containing all elements in both A and B
	- [[Relative Complement]] - Also known as set subtraction
		- Set containing all elements of A not in B
		- $B \backslash A \ne A \backslash B$
	- [[Absolute Complement]]
		- Need a universal set in problem
		- Absolute complement of set A are all elements in the universe not in set A
- ![[Pasted image 20230601084932.png|500]]  
	- [[Symmetric Difference]] - $A \Delta B$
		- The set containing elements that are in the union of A and B but not in the intersection of A and B
