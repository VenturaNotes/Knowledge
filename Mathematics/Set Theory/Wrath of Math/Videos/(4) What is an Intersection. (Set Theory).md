	- Intersection
		- A set that contains the intersection of 2 sets will only have elements that are an element of A and an element of B.
- Example
	- ![[Screenshot 2023-01-10 at 11.58.14 AM.png]]
		- Set A
			- All positive even numbers less than or equal to 12
		- Set B
			- All positive multiples of 3 less than or equal to 12
