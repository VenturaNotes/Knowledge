- Write the $1^{st}$ 3 natural numbers
	- {1, 2, 3}
		- Use curly brackets to mark start and end of a set
		- Inside the brackets we have elements
		- Each of the 3 numbers is an element (or a member)
		- Use commas to separate the elements
		- Method above is known as <mark style="background: #FFF3A3A6;">listing</mark> or [[Roster Method|roster method]]
