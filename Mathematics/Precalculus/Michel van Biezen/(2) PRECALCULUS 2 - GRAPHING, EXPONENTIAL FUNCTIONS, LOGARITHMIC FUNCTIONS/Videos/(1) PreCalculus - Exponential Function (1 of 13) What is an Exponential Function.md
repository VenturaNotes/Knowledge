	- An exponential function is a function where the independent variable is the exponent
		- Standard Form
			- $y = c^x$
				- c is the base (constant)
				- x is the exponent (variable)
		- General Form
			- $y = c^{f(x)}$
				- Sometimes it could be to the function of x (f(x))
	- t for time is an independent variable
