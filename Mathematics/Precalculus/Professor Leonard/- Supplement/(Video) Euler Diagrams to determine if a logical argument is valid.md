- Asked in a logic class to determine if an argument is valid or not
- Create Euler diagrams to test validity of an argument.
	- Use these for arguments that have qualified statements in it
		- Qualified meaning it has words like all, or some, or none in them.
- Example (First 2 statements are premises)
	- All math teachers love coffee (premise)
	- Angie is a math teacher (premise)
	- ============= (therefore)
	- Angie loves coffee. (conclusion)
