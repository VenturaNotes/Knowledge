- Solving [[linear equation|linear equations]] in one variable
- A linear equation in one variable:
	- ax + b = c
		- a, b, and c are real numbers
		- a $\neq$ 0
- [[Addition Property of Equality]]
	- Tells us we can add/subtract any number to/from both sides of an equation
	- ![[Screenshot 2023-01-07 at 2.16.09 PM.png]]
- [[Multiplication property of equality]]
	- Tells us we can multiply/divide both sides of an equation by the same non-zero number
	- ![[Screenshot 2023-01-07 at 2.16.58 PM.png]]
- Solving a Linear Equation in one Variable:
	- Simplify each side completely
	- Isolate the term that includes the variable on one side of the equation
	- Isolate the variable
	- Check the answer
