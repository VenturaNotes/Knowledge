- [[Expanded Notation]]
- Lesson Objectives:
	- Learn how to write a number in expanded notation
	- Learn the difference between [[Standard Notation|standard notation]] and expanded notation
- In expanded notation, we're writing a number in a way that emphasizes the place value for each digit in that number
- Place value chart find the place value for each digit in the number
