## Source [^1]
- In a Markov chain, the `ith` state has period $d$ where $$d = gcd\{n : [M^n]_{ii}>0\},$$where M denotes the transition matrix. So in the two state Markov chain with $$M = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$$each state has period 2. A state is aperiodic if $d=1$.
