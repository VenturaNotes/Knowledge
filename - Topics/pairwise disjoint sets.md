## Source [^1]
- A collection of sets is pairwise disjoint if any two sets in the collection are disjoint
	- Examples
		- P = {{1}, {2, 3}, {4, 5, 6}} is a pairwise disjoint set
		- P = {{1, 2}, {2, 3}} is not pairwise disjoint
			- 2 is a common element in two sets
