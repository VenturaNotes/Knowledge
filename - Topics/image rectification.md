## Source [^1]
- Mathematical techniques for merging images of the same scene taken from different vantage points into a single image. The process depends on being able to identify the positions of the same visual features in each image, so that an appropriate transform can be derived. In geographical information systems, image rectification is often used to convert aerial photographs into orthophotos.
