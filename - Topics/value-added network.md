## Source[^2]
- A value-added network (VAN) is used by businesses to exchange information with each other. A VAN is a private network with added benefits through which value-added carriers provide value-added services, such as secure email, message encryption and management reporting. A VAN's main goal is to facilitate electronic data interchange (EDI) between businesses.
