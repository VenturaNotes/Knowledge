## Source [^1]

## Source[^2]
- n. 
- (1) A belief that one is recalling events or experiences that never really occurred. See also EXPERIMENTALLY INDUCED FALSE MEMORY, FALSE MEMORY, FALSE MEMORY SYNDROME, REALITY MONITORING. 
- (2) Any disorder of memory involving misremembering rather than failing to remember, including déjà vu and jamais vu. \[From Greek para beside + mnasthai to remember + -ia indicating a condition or quality]
