## Source [^1]
- A system built into many electronic devices, such as smartphones, in which a sensor is used to adjust the screen brightness according to the measured ambient light conditions. This enables the screen to be dimmed wherever possible, saving power and enhancing battery life, while still allowing it to be as readable as possible in bright light.
