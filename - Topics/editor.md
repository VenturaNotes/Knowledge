## Source [^1]
- A software application that facilitates the editing of a type of data (text, image, video, audio, etc.). See also LINK EDITOR, TEXT EDITOR.
## Source[^2]
- A type of program that allows users to create and modify text files. Common uses are editing program source code and hypertext markup language (HTML) files.
