## Source [^1]
- (1) An array of bits that map one to one to the monochrome image on a raster display. If a color or grey-level image is required, needing many bits to define each pixel in the display, a pixmap is required. 
- (2) A data structure consisting of an array of bits. 
- (3) In popular but incorrect usage, a pixmap.
