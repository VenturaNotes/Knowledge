## Source [^1]
- An image on a web page that can function as a hyperlink. The term is particularly used for an image in which different parts give different links. For instance, clicking on different rooms in a plan of a house might link to photographs of each room. Originally image maps were executed on the web server but they are now handled in most browsers.
