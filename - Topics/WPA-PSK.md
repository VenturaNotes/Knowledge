## Synthesis
- This is a mode of wireless network security where all devices connecting to a network use the same shared password ([[passphrase]]) to authenticate and access the network.
	- Commonly used in home and small business settings
	- Considered a more secure option compared to older Wi-Fi encryption methods like [[Wired Equivalent Privacy|WEP]]
