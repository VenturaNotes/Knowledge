## Source [^1]
- In an amplifier, the gain for small signals is generally not maintained for large input signals but is reduced; this is called gain compression. This nonlinearity is quantified as the input signal level for which the output signal is one decibel (dB) below the linear gain value; this is the 1 dB compression point (see diagram).
