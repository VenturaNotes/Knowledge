## Source [^1]
- (1) One of the basic actions performed on sets that, when applied in the form$$insert(el, S )$$adds the element $e l$ to the set $S$. If $e l$ is already in $S$ the operation has no effect on the membership of $S$. See also OPERATIONS ON SETS. 
- (2) One of the basic actions performed on lists, that places a new element into a list, not necessarily at one end or the other.
