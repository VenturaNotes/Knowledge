## Source [^1]
- A file containing data, such as a file created within an applications program; for example, it may be a word-processing document, a spreadsheet, a database file, or a chart. Data files are normally organized as sets of records with one or more associated access methods. Compare PROGRAM FILE.
## Source[^2]
- A file on a computer system that contains data, as opposed to one that contains a program. A data file is usually subdivided into records and fields.
