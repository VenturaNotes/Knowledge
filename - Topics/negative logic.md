## Source [^1]
- (1) A logic system in which the normal meanings of the binary signal levels are interchanged, e.g. high voltage equals logic 0 , low voltage equals logic 1.
- (2) (complementary logic) A logic system in which all the Boolean variables and Boolean functions behave as though they were complements.
- Compare POSITIVE LOGIC.
