## Synthesis
- The process of paying down that debt by improving code structure without changing what it does, making it cleaner, more efficient and easier to maintain.
## Source [^1]
- The process of restructuring existing computer code without changing its external behavior, aimed at improving nonfunctional attributes of the software.
