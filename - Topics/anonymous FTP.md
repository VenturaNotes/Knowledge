## Source [^1]
- A mechanism that allows a user to take copies of files from a network file server, without having a named account on the server. The user 'anonymous' is allowed to log into the server, and has limited privileges allowing him or her to initiate an outgoing file transfer. See also FILE TRANSFER PROTOCOL.
