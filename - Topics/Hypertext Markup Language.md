## Source [^1]
- Hypertext Markup Language (better known as HTML) is a document-formatting language that is used for the creation of Web pages and online apps. Essentially, it is a code that allows authors to dictate the structure of a Web page and how it is displayed in a web browser. Some of the most common uses of HTML include publishing online documents (blog posts, for example) that contain headings, photos, or tables. Additionally, HTML is used to create forms that enable online transactions to occur, and also provides the ability to add videos, sounds, and spreadsheets throughout a document.
## Source[^2]
- (HTML) A widely used markup language for structuring documents on the Internet. Text and media within a document are marked by a series of 'tags' to denote their structural relationship with other parts of the document and other documents on the Internet. For example, Chapter One specifies that the text "Chapter One" is a first-level heading within the document. The presentation of the text can be set within the tags, for example font size, but should normally be specified by means of a collection of presentation styles to allow for greater accessibility of the document to those with disabilities. In addition there are a number of programming notations that can be embedded in, or used to generate, HTML to facilitate advanced data handling.
