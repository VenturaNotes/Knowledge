## Source [^1]
- (symbol: $F$) In a given electrical component, circuit, or system, a measure of the additional noise contributed by that component, circuit, or system. It can be defined as the ratio of the signal-to-noise ratio at input to the signal-to-noise ratio at output:$$F = \frac {S _ {\mathrm {i}} / N _ {\mathrm {i}}}{S _ {\mathrm {o}} / N _ {\mathrm {o}}}$$If the gain of the system is given by $G$, this can be written$$F = \frac {S _ {\mathrm {i}} / N _ {\mathrm {i}}}{G S _ {\mathrm {i}} / \left(G N _ {\mathrm {i}} + N _ {\mathrm {a d d}}\right)} = \frac {N _ {\mathrm {i}} + \left(N _ {\mathrm {a d d}} / G\right)}{N _ {\mathrm {i}}}$$
