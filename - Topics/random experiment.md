## Synthesis
- Two key features of a random experiment
	- Possible outcome $\gt$ 1
	- Not possible to predict outcome in advance
## Source [^1]
- An activity that produces a result or an outcome is called an experiment. It is an element of uncertainty as to which one of these occurs when we perform an activity or experiment.
- When an experiment satisfies the following two conditions, it is called a random experiment
