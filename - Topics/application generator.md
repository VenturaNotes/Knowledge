## Source [^1]
- A computer program$\textemdash$a software tool$\textemdash$that is capable of creating a range of application programs in a particular domain. The generated program will be configured by information provided by the person using the application generator. Domains in which application generators are frequently encountered include simulation, process control, and user interface software. See also FOURTH-GENERATION LANGUAGE.
