## Synthesis
- A web browser uses the HTTP protocol at the [[application layer]] to fetch web pages
- Accesses the internet ([[Chrome]])
## Source[^1]
- Programs like [[Internet Explorer]] that act as navigable windows on the web is a web browser
## Source[^2]
- An application used to view and interact with web pages and to navigate between them. Technically it is a client application that implements HTML and related standards and which communicates with servers over a TCP/IP network using HTTP. Common browsers in use today include Internet Explorer, Chrome, Firefox, and Safari.
## Source[^3]
- A software application, such as Microsoft Internet Explorer or Google Chrome, that provides an easy method of accessing, viewing, and interacting with information stored as HTML web documents on different web servers.
