## Synthesis
### Printing Variables
```python
name = "Alice"
age = 30

print(f"My name is {name} and I am {age} years old.")
```
### Adding Multiple Lines
- These are [[formatted string literal (Python)|formatted string literals]]
```python
# Single f-string
print(f"Hello World \nThis is a new line.")

# Multiple f-strings
print(f"Hello World \n" 
	  f"This is a new line.")
	  
""" Output
Hello World 
This is a new line.
"""
```
