## Source [^1]
- (routing control plane) The part of a network responsible for implementing routing protocols and configuring the routing data used by the packet switching or data plane part of the network. Although both planes are typically implemented in hardware, there is a growing trend toward software-defined networking (SDN) in which the control plane is implemented using general-purpose computing devices with protocols and configuration managed by software.
