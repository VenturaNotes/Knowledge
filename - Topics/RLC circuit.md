## Source [^1]
- (RLC filter) A circuit built using resistors, capacitors, and inductors. The simplest are two-port networks that have one of each component, connected either in series with the input port or in parallel with the output. These can be used to produce filters with better performance than simpler RC or RL filters, as well as band-pass and band-stop filters. See also RC CIRCUIT; RL CIRCUIT.
