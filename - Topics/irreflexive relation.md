## Source [^1]
- A relation $R$ defined on a set $S$ and having the property that $x R x$ does not hold for any $x$ in the set $S$. Examples are 'is son of', defined on the set of people, and 'less than', defined on the integers. Compare REFLEXIVE RELATION.
