## Source [^1]
- In a graphical user interface (GUI), the pattern on those parts of the screen outside the currently displayed windows. The wallpaper can be chosen from those supplied with the GUI or can be provided by the users. Small repeated patterns or screenfilling pictures can be used. Compare SCREENSAVER.
