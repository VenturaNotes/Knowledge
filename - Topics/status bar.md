## Source [^1]
- In many kinds of display, including text and graphical user interfaces, a line of usually textual information about current status. A word-processor status bar may tell the user what line and page is currently in use, if typeover or insert mode is in force, and which font is currently selected, while a database status bar might have the current record number and field name and whether the display may be edited.
