## Source[^2]
- Internet protocol
	- Any protocol that allows traffic to pass between networks. The most widely used is IP, the Internet Protocol. See also INTERNETWORKING.
- IP (Internet Protocol) The basic communications protocol in TCP/IP networks for relaying data across network boundaries. Its routing function allows internetworking, and is fundamental to the establishment of the Internet.
