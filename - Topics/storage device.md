## Source [^1]
- A device that can receive data and retain it for subsequent retrieval. Such devices cover a wide range of capacities and speeds of access. The semiconductor devices used as the main memory for the processor may take only a few nanoseconds to retrieve data but the cost of storing each bit is very high by comparison with the devices used as backing store, which may take milliseconds or even many seconds to retrieve data. See also MEMORY HIERARCHY, MEMORY MANAGEMENT.

## Source[^2]
- Any device that holds information by a physical or chemical method. The term is particularly applied to a memory in a computer, which holds information usually in binary form.
