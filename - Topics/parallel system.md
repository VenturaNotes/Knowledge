## Source [^1]
- A system in which multiple devices, each with a single input and output, are connected side by side so that all the inputs are joined together, as are all the outputs (see PARALLEL). Devices can range from simple circuit elements to more complex signal-processing or data-transformation elements. Compare SERIES SYSTEM.
