## Source [^1]
- A sophisticated hypothesis test which compares the likelihood of the observed value happening for different possible values of the parameter under investigation. In its simplest form, the comparison is between two possible values, but more generally the alternatives are ranges of the parameter value and the comparison is for maximum likelihoods in the interval.
