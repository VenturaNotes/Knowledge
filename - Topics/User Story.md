## Synthesis
- A description of a feature desired by a user or customer of the system. For example, they might want to be able to log in to access their account.
	- #question Are there other ways aside from logging in to access an account?
	- #question How are users and customers of the system different
## Source [^1]
- An informal, natural language description of one or more features of a software system, often used in Agile software development methodologies to capture requirements from an end-user perspective.
