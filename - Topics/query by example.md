## Source [^1]
- (1) A screen-based form-filling user interface to a relational database. The user fills in visual tables with conditions and examples, and the appropriate database query is constructed and run behind the scenes. 
- (2) (in information retrieval) An interface in which the user submits examples and the software performs a search based on some measure of similarity.
