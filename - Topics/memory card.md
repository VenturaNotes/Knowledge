## Source[^2]
- A small flash memory storage device in the form of a plastic card with terminals along one edge. Most digital cameras and smartphones have slots for memory cards that can be used to store photographs and videos. While there are a number of differing standards for memory cards, the most successful one currently is the Secure Digital (SD) standard. Commercially available memory cards can have capacities measured in hundreds of gigabytes. See also USB FLASH DRIVE.
