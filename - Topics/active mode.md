## Source [^1]
- The operating state of a bipolar junction transistor in which the current between emitter and collector varies in rough proportion to changes in the base current. This occurs when the base voltage is between the collector and emitter voltages, with enough difference between base and emitter voltages to ensure that the base-emitter pn junction is turned on.
