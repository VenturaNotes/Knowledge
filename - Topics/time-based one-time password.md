## Synthesis
- A security method that generates a unique, temporary password based on the current time, acting as an extra layer of authentication where users need to input this code alongside their regular password to access an account.
- The code automatically updates at regular intervals (usually every 30 seconds) to prevent unauthorized access
## Source [^1]
- A common form of [[two-factor authentication|2FA]]. Unique numeric passwords are generated with a standardized algorithm that uses the current time as an input
