## Source [^1]
- A good that is available only if a particular event (or 'state of the world') occurs, for example an ice cream delivered only if the sun shines. Contingent commodities form the basis of general equilibrium models of uncertainty, and can be used to model financial markets since a financial security can be modeled as delivering a bundle of contingent commodities. See also Arrow-Debreu eConomy.
