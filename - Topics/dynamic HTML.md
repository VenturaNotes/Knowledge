## Source [^1]
- An extension of HTML allowing a Web page to change in response to the user, without downloads from the server. Dynamic HTML involves scripts written in Javascript or VBScript and embedded in the HTML document. An example of its use is in displaying menus when the cursor is positioned at a particular place on the page.
