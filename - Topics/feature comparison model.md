## Source [^1]
- $n$. Any theory of concept formation according to which a decision as to whether a particular item or element belongs to a class proceeds by comparing the features (1) of the item with those of the class. For example, the theory posits that a person decides whether a particular animal is a cat by checking it for various features associated with cats, such as whiskers, a tail, a propensity to mew, and so on.
