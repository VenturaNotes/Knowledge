## Source [^1]
- Convert the number 3 and imaginary number 5 into a complex number:
	- `x = complex (3, 5)`
		- Output would be `(3+5j)`
- This function returns a [[Complex numbers|complex number]] by specifying a real number and an imaginary number
- Syntax
	- `complex(real, imaginary)`
- Parameter Values
	- real
		- Required. Default is 0. The real number can also be a string such as '3+5j'. Then the second parameter should be omitted
	- imaginary
		- Optional. Number representing imaginary part of complex number. Default 0
- Converting string to a complex number
	- `x = complex('3+5j')`
		- Output would be `(3+5j)`
			- This includes the parentheses
