## Source [^1]
- A set towards which a dynamical system tends to evolve or remain close. So a stable equilibrium is an attractor, but more generally an attractor can be a point, or a finite set of points, or a curve, surface, or higher-dimensional manifolds. See LORENZ ATTRACTOR.
