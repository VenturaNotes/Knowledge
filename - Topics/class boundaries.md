## Source [^1]
- The boundaries for the classes when data is grouped. For continuous data, measurements will normally be recorded initially to some degree of accuracy. For example, if intervals are $1.5 \le x < 2.0; 2.0 \le x < 2.4 \dots$ with observations recorded correct to the nearest $0.1$, then the class boundaries are $1.45, 1.95, 2.45$, etc.
