## Source [^1]
- (RAM drive, virtual disk) A portion of a computer's RAM memory made to behave as though it were a disk. It can be written to and read from as though it were a real disk but it is very much faster. However, it will generally have a lower capacity than a hard disk. The contents of the RAM disk are not preserved when the computer is switched off. RAM disks are not widely used: disk caches that improve the performance of real disks are a better use of RAM.
## Source[^2]
- A portion of ROM that has been set up to simulate a disk drive. The access of files on the RAM disk is equivalent to the access on a hard disk but the access time is significantly faster.
