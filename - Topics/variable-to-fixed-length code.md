## Source [^1]
- A code in which the data input to the encoder may vary in length between iterations of the encoding algorithm, but the data output is always of the same length. The decoding of a VTFL code is FTVL. Also, VTFL is the decoding scheme for data that were encoded by a FTVL code (e.g. a Huffman code).
