## Source [^1]
- A technique whereby a list of names and addresses can be merged with a form letter to produce a set of personalized letters to a number of people, known as a mailshot. The same list can also be used to produce address labels. The technique is quite general and can be used wherever a list of items is to be printed or displayed in a number of different ways.
