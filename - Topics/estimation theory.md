## Source [^1]
- Statistical techniques for estimating key system parameters from samples of data taken from that system, assuming that the data has some randomness. Usually there is some theoretical system model that describes how the parameters will affect the observed data, and estimation theory is used to test how well the system theory corresponds to actual measured data.
