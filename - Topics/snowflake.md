## Source [^1]
- Informal A network made up of nodes that can each support a number of enduser devices, with the nodes themselves connected in an arbitrary configuration. A number of Ethernet hubs each supporting end-user systems connected by twisted pairs, with the hubs interconnected by coaxial cable or optical fibre, might be described as a snowflake.
