## Source [^1]
- An interpreted object-oriented programming language that includes features of both imperative and functional languages. Created by Yukihiro Matsumoto and released in 1995, it was influenced by such languages as Perl, Smalltalk, Eiffel, Ada, and Lisp. Its combination of elegance and apparent simplicity with underlying power has gained Ruby an steadily increasing popularity, especially through the Ruby On Rails web application framework.
