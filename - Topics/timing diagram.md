## Source [^1]
- A graphical description of the operation of a sequential circuit; the state(s) of all the relevant variables (inputs, internal memory, and outputs) are shown as functions along the time dimension.
## Source[^2]
- A diagram that shows the relative timing of a number of input and/or output lines in a logic circuit. Timing diagrams are used particularly when connecting devices together that require some cooperation, such as memory devices and a microprocessor.
