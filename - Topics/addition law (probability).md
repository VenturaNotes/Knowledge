## Source [^1]
- If A and B are two events then the addition law states that the Pr(A or B)=Pr(A) + Pr(B) - Pr(A and B), or using set notation $P(A \cup B) = P(A) + P(B) - P(A \cap B)$. In the special case where A and B are mutually exclusive events this reduces to $P(A \cup B) = P(A) + P(B)$
