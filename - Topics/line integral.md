## Source [^1]
- An integral along a curve. Say a curve $C$ is parametrized by $\mathbf{r}(t)$ where $a \leq t \leq b$. For a scalar function $f$, defined on $C$, we define the line integral$$\int_C f \, ds = \int_a^b f(\mathbf{r}(t)) \left| \frac{d\mathbf{r}}{dt} \right| dt$$where $s$ denotes arc length. It is independent of the parametrization. If $f$ represents the density of a wire in the shape of $C$, then the line integral would equal the mass of the wire.
- For a vector function $\mathbf{F}$, defined on $C$, we define the line integral$$\int_C \mathbf{F} \cdot d\mathbf{r} = \int_a^b \mathbf{F}(\mathbf{r}(t)) \cdot \frac{d\mathbf{r}}{dt} dt$$which equals the work done by $\mathbf{F}$ along $C$. It is independent of the parametrization up to sign but changes sign if the orientation of the parametrization is reversed.
- Line integrals are also called path integrals, particularly in complex analysis.
