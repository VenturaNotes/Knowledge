## Source [^1]
- Any process in which the variables that define the state of the system can be made to change in such a way that they pass through the same values in the reverse order when the process is reversed. It is also a condition of a reversible process that any exchanges of energy, work, or matter with the surroundings should be reversed in direction and order when the process is reversed. Any process that does not comply with these conditions when it is reversed is said to be an irreversible process. All natural processes are irreversible, although some processes can be made to approach closely to a reversible process.
