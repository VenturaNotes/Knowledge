## Source [^1]
- A square matrix in which each column contains precisely one nonzero element, which is equal to unity. If $P$ is an $n$'$n$ permutation matrix and $\mathbf{x}$ is a vector of $n$ elements, the vector $P \mathbf{x}$ will be a permutation of the elements of $\mathbf{x}$.
