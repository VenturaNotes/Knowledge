## Source [^1]
- An iterable is an object that can return its members one at a time, allowing you to loop through them. Think of it as a collection of items where you can access each item sequentially
- Examples:
	- Lists
	- Tuples
	- Strings
	- Dictionaries
- An iterable is an object that can be looped over or iterated through. This means you can access its elements one by one in a sequence
