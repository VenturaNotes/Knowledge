## Source [^1]
- (grounded-emitter connection) A method of operating a bipolar junction transistor in which the emitter is common to both the input and output circuits and is usually earthed (see diagram). The base is used as the input terminal and the collector as the output terminal.
- The equivalent connection for a field-effect transistor is common-source connection and for a thermionic valve is common-cathode connection.
