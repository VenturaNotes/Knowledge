## Source [^1]
- A data network that is accessible for use by private individuals, or by organizations other than that operating the network. In most countries it is necessary to be licensed to operate a PDN. See also PTT, PUBLIC NETWORK OPERATOR, PUBLIC TELECOMMUNICATIONS OPERATOR.
