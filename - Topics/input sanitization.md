## Synthesis
- Input sanitization is the process of cleaning or filtering user input data to remove potentially harmful characters or code, ensuring that only safe data is processed by an application, thus preventing security vulnerabilities like SQL injection and cross-site scripting (XSS) attacks by removing malicious code from user-submitted data
## Source [^1]
- The process of modifying or removing potentially harmful data entered by users to prevent web-based attacks like [[SQL injection]] and cross-site scripting ([[Cross-Site Scripting|XSS]])
