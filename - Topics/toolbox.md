## Source [^1]
- toolbox A set of software tools, probably from several vendors, not necessarily as closely related or providing as full coverage of the  software life cycle as a [[toolkit]]. The set of tools in a toolkit is usually from a single vendor. See also CASE, PSE, SOFTWARE ENGINEERING ENVIRONMENT.
