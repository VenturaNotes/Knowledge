## Source [^1]
- The operation code of a particular machine, and hence, by association, code specific to a particular machine.
## Source[^2]
- The binary code used to represent the instruction set of a particular computer. The machine-code instruction set is defined by the designer of the computer and constrained by the architecture chosen.
