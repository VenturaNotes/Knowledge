## Source [^1]
- A program that translates assembly code into machine code.
## Source[^2]
- An assembler is a computer program that takes in basic instructions and converts them into a pattern of bits that the computer's processor can use to perform basic operations.

## Source[^3]
- (1) A program that takes as input a program written in assembly language and translates it into machine code or relocatable code. 
- (2) Colloquial An assembly language.
