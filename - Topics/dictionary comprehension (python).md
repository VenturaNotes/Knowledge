## Synthesis
- Creates a dictionary by mapping keys to values based on an iterable
```python
names = ["Alice", "Bob", "Charlie"]
name_lengths = {name: len(name) for name in names}  

# Output: {'Alice': 5, 'Bob': 3, 'Charlie': 7}
```
