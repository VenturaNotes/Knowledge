## Source [^1]
- A figure-of-merit for a computer system in which some description of operating rate such as instructions per minute, jobs per day, etc., is used. It is a measure of how much work gets done in a given time interval. See also BANDWIDTH, LATENCY.
## Source[^2]
- Throughput is a computing term that measures the number of units a network processes within a specific period of time, typically an hour. SMBs can use this calculation to determine the volume of work or information that flows through a computer system and how much data transfers from one location to another.
## Source[^3]
- A measure of the number of distinct signals or data packets that a system can process in a given time. It is typically measured by counting arriving signals or packets at their destination over a fixed time interval. See also LATENCY.
