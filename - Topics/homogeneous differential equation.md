## Source [^1]
- (1) A linear differential equation that has the property that whenever $f(x)$ is a solution, then so is $c f(x)$, where $c$ is an arbitrary non-zero constant. 
- (2) A first-order differential equation in which the coefficients are all homogeneous functions. A homogeneous function is a function $f$ where $f(k x) = k^i f(x)$ for arbitrary real $k$ and fixed $i$.
