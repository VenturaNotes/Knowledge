## Source [^1]
- (true complement) For an integer represented in a fixed-radix number system, a number formed by adding one to the radix-minus-one complement of the given integer. For example, in the decimal system the ten's complement of 0372 is 9628 (i.e. $9627+1$ ); in the binary system the two's complement of 1100 is 0100 (i.e. $0011+1$ ). See also COMPLEMENT NUMBER SYSTEM.
