## Source [^1]
- In a communication system, the use of a second channel or path that can be used if the primary channel becomes unavailable for any reason. For example, in a telephone connection between two subscribers there are usually a number of different routes that can be used to make the connection. The route selected depends on the level of usage of the system as a whole and whether any physical damage exists on any of the routes.
