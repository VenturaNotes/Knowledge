## Source [^1]
- An image produced by combining many images of the same object, each taken in a different narrow wavelength band. These bands are contiguous, so that they cover the full spectrum between the lowest and highest wavelengths. This enables the determination of an individual spectrum for each image in the combined image. Spectral analysis techniques can thus be used to classify the materials present in the surface of the object at the corresponding point. See also MULTISPECTRAL IMAGE.
