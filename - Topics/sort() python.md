## Synthesis
- Will sort pairs in a list in ascending order
- Given `A = [[3, 0], [1, 1], [4, 2], [2, 3]]`
- `A.sort()` will do `A = [[1, 1], [2, 3], [3, 0], [4, 2]]`
	- This will change the value of `A` itself without needing an assignment operation
