## Source [^1]
- A modeling and design technique for control systems that defines an explicit state space and the dynamic relationship between it and the system inputs, outputs, and future state changes. Both the output and future state changes are defined as functions of the current input and state.$$\text{output} = f_{\mathrm{o}} (\text{input}, \text{state})$$$$\text{state}' = f_{\mathrm{s}} (\text{input}, \text{state})$$
- Also specified are the starting state of the system, and any final states, if it is designed to terminate. See also FINITE-STATE MACHINE.
