## Source [^1]
- The human-readable instructions that a programmer writes—often in a high-level programming language—before it is compiled or interpreted into machine language.
## Source[^2]
- The form of a program that is input to a compiler or translator for conversion into equivalent object code. See also CODE.
## Source[^3]
- The language in which programmers write their programs, though it cannot be directly understood by a computer. Either the source program (code) is converted to an object code program using a compiler or it is run one line at a time using an interpreter.
