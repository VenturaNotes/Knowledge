## Source [^1]
- of a string of symbols, $a_{1} a_{2} \ldots a_{n}$. Any string of symbols of the form$$

\begin{aligned}

& a_{i} a_{i+1} \ldots a_{j} \\

& \text {where } 1 \leq i \leq j \leq n

\end{aligned}

$$The empty string is regarded as a substring of any string.
