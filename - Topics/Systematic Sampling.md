## Source [^1]
- A sampling procedure in which every $k$th element is selected from the frame, after a random start somewhere within the first $k$ elements, where $k=$ the sampling interval. For example, an alphabetical list of first-year students might be obtained from a college and every fifth student selected for the sample.
