## Source [^1]
- A sequence of positive integers can be recursively defined as follows: $x_{n+1}=3x_n+1 \text{ if } x_n \text{ is odd and } x_{n+1} = x_n/2 \text{ if } x_n \text{ is even}$. The Collatz conjecture states that that sequence eventually returns to 1 from any initial terms $x_1$, e.g. 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1. The conjecture is unproven but has been verified for initial terms up to $10^{20}$
