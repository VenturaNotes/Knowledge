## Source [^1]
- A form of scalable outline font originally developed by Apple, and now used extensively on almost all personal computers. TrueType fonts have outlines based on quadratic B-splines, and a single file contains all the information necessary to print and display the font on the screen and to produce bold and italic versions. See also OpenType FONT, POSTSCRIPT FONT.
