## Synthesis
### Description
- A contiguous array (or subarray) is a sequence of elements that are next to each other in the original array, with no skipping or gaps between them
### Example
- Given `[10, 20, 30, 40, 50]`
	- Contiguous subarray
		- `[20, 30, 40]`
			- Elements in order of original array
	- Not contiguous subarray
		- `[10, 30, 40]`
			- Elements not in order of original array
