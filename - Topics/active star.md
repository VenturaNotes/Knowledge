## Source [^1]
- A network topology in which the outer nodes connect to a single central node that processes all messages in the network, including messages that it forwards from one outer node to another. A failure of the central node causes the entire network to fail. See also NETWORK ARCHITECTURE, PASSIVE STAR, STAR NETWORK.
