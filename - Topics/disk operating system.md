## Source [^1]
- (DOS) A computer operating system that resides on a hard disk drive. It originated from a period in which hard disk drives and floppy disks were optional components of computers. The first widely used DOS in personal computers was PC-DOS, developed for IBM by the Microsoft Corporation. Subsequently Microsoft marketed an almost identical version under the name MS-DOS.
