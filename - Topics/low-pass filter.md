## Source [^1]
- A filtering device that permits only those components in the frequency domain whose frequencies lie below some critical value to pass through with little attenuation, all other components being highly attenuated. See also BAND-PASS FILTER, BAND-STOP FILTER, HIGH-PASS FILTER.
