## Source [^1]
- In nonlinear systems and devices, the current gain that occurs where the input signal is large enough to affect the operating point, so that the ratio of output current to input current varies according to the value of the input. This should be contrasted with the small signal current gain, which occurs where the signal is not large enough to have this effect and the ratio remains constant.
