## Source [^1]
- The function defined by $\Gamma(x) = \int_0^{\infty}t^{x-1}e^{-t}dt$ for $x > 0$. Integration by parts yields $\Gamma(x+1) = x\Gamma (x)$, and $\Gamma(1) = 1$ so if $n$ is a positive integer $\Gamma(n) = (n-1)!.$
