## Source [^1]
- Television services that are transmitted from the content provider to the viewer using the Internet. Content is viewed using either a web-browser on a personal computer, smartphone, or smart TV, or via an Internet-capable set-top box into a standard television. A variant, IPTV, is a closed system that uses IP to route data to the customer; it therefore bypasses the public Internet, making it easier to give guarantees regarding bandwidth.
