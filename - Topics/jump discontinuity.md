## Source [^1]
- A point at which a function f(x) has both left-hand and right-hand limits but the limits are not equal. For example, $f(x) = -1$ if $x < 0$ and $f(x) = 1$ if $x > 0, f$ has a jump discontinuity at $x = 0$
