## Source [^1]
- A method of encoding Unicode code points using one-byte unsigned integers; from one to four such integers are used depending on the code point. UTF-8 is the most widely used Unicode encoding scheme and is the default encoding for XML documents.
- http://www.unicode.org/versions/Unicode5.0.0/ch03.pdf
	- Details of UTF-8 encoding from the Unicode (version 5) standard
