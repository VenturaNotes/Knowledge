## Source [^1]
- A set of polynomials $\{p_k(x)\}$, $k \ge 0$ are said to be orthogonal over an interval $[a,b]$ if for some positive weighing function $w(x)$, $$\int_a^bw(x)p_i(x)p_j(x)dx = 0$$when $i \ne j$. See Chebyshev polynomials, Hermite polynomial, Legendre polynomials
