## Synthesis
- Counterfactual logic is associated with philosopher [[David Lewis]]
	- #question Who is David Lewis?
### Description
- A conditional statement that discusses what would have happened under different circumstances. 
- Separate from "material conditionals" used in math or logic
	- Counterfactuals deal with subjunctive possibilities
		- #question What is a subjunctive possibility?
		- These are scenarios where the antecedent is known to be false in the actual world
- For example
	- In standard logic, $P \to Q$ is vacuously true when P is false
	- In counterfactual, we care about the causal or logical between the "what if" and "then"
		- #question So how would this change the standard logic $P \to Q$?
		- #question So do we say 'What if P, then Q', instead of 'if P, then Q' for the counterfactual conditional?
- Structure of counterfactual
	- Antecedent $(P)$: A condition that did not happen
		- "If I had caught the train."
	- Consequent $(Q)$: The imagined outcome
		- "I would have been on time"
### Example
- Counterfactual: "If I had struck that vase with a hammer, it would have broken."
	- In standard logic using a truth table, anytime $P$ is false, the statement would be true.
	- In counterfactual logic, we evaluate this by imagining the "Closest Possible World"
		- This typically means that in a world almost identical to ours (except we picked up the hammer), the vase breaks. So the counterfactual is considered True. 
- If we're given a statement "If I had struck that vase with a hammer, it would have turned into a bird"
	- In standard logic using a truth table, the statement would still be true anytime $P$ is false
	- In counterfactual logic, this statement would be false, because in the closest possible world where you hit a vase, physics still applies.

### Worked Problem (Material Condition vs Counterfactual Condition)
- Scenario
	- Fact A: I did not go to the store
	- Fact B: I do not have milk
#### Material Conditional ($P \to Q$)
- Statement: "If I went to the store, then I have milk."
	- Logic: Since I did not go to the store, the P is False
	- Result: Vacuously True. Logic holds because `if` never happened
#### Counterfactual Conditional $P~\Box\kern-0.4em\rightarrow Q$
- Statement: "If I had went to the store, I would have had milk"
- Closest Possible World
	- Case 1: Store is open and stocked with milk (result true)
	- Case 2: Store is permanently closed or out of milk (result False)
- Conclusion: Counterfactual requires a real-world (or possible-world) connection between action and result 
### Difference Between Material Condition & Counterfactual

| Feature          | Material Conditional $(\to)$            | Counterfactual $(\Box\kern-0.4em\rightarrow)$ |
| ---------------- | --------------------------------------- | --------------------------------------------- |
| Focus            | Truth values of $P$ and $Q$.            | Causal/Logical connection.                    |
| False Antecedent | Always results in a **True** statement. | Must be evaluated based on Possible Worlds.   |
| Typical Phrasing | "If $P$, then $Q$."                     | "If $P$ had been, $Q$ would have been."       |
