## Source [^1]
- $n$. The violet cone pigment in the retina of the eye that was the first cone pigment studied chemically; also, a generic name for any cone pigment, though this sense is avoided in careful usage. Also called visual violet. \[From iodine + Greek ops an eye + -ine indicating an organic compound, so called because the element iodine is violet in color]
