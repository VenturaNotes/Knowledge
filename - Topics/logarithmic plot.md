## Source [^1]
- A graph of a function in which one or both axes are scaled logarithmically. Such plots are useful for displaying information where there is a large variation in the size of the values involved. A semi-logarithmic plot is one in which only one axis is scaled logarithmically, while a log-log plot has both axes scaled in this way.
