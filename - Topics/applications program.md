## Source [^1]
- (application program, app) Any program that is specific to a particular role and makes a direct contribution to performing that role. For example, where a computer handles a company's finances a payroll program would be an applications program. By contrast, an operating system or a software tool may both be essential to the effective use of the computer system, but neither makes a direct contribution to meeting the end-user's eventual needs.
