## Source [^1]
- A system for locating someone inside a building by tracking their wireless mobile devices. This is achieved by triangulating, using radio sources such as WiFi and Bluetooth, and by getting access to inertial measurement unit data from the devices. GPS is generally ineffective in indoor environments.
