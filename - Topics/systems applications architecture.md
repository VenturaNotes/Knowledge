## Source [^1]
- IBM's family of standard interfaces that enable software to be written independently of hardware and operating systems. Published in the 1980s, the only part that is still important is CUA ([[Common User Access]]). This sought to standardize user interfaces and has influenced OS/2, Windows, and Java GUI packages
