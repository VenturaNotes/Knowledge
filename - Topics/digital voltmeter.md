## Source [^1]
- (DVM) A voltmeter that displays the values of the voltage as numbers. The input is usually supplied as an analogue signal and this is repetitively sampled by the voltmeter, which displays the instantaneous values. The instrument thus functions as a type of analogue-to-digital converter.
