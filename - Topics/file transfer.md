## Source [^1]
- The act or process of transferring a file from one computer system to another, usually but not necessarily across a network. The problems associated with a file transfer between two identical computer systems are minimal, but they increase when the systems use different hardware and run under different operating systems. See also FILE TRANSFER PROTOCOL.
