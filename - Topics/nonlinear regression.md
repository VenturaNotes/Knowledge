## Source [^1]
- A regression in which the variable of interest is modeled as a function of explanatory variables that is nonlinear in the model parameters and that cannot be converted into a linear form by transformation of the variables. A nonlinear regression can be estimated by nonlinear least squares if the regression function is continuously differentiable in the parameters.
