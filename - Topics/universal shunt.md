## Source [^1]
- A galvanometer shunt that is tapped so that it can pass designated fractions (0.1, 0.01, etc.) of the main current and can thus be used with galvanometers of widely varying internal resistance. An example is the Ayrton shunt, which is a relatively large resistance with suitable tappings.
