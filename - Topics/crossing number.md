## Source [^1]
- A knot in three dimensions can be represented, as if laid flat, with various over- and under-crossings. There is a minimum number of crossings for how a given knot can be so represented, for example 3 for the trefoil. This minimum crossing number is invariant under isotopy. See knot theory.
