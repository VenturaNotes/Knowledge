## Source [^1]
- $n$. In psychometrics, a consistent tendency to respond to yes/no questionnaire items by answering Yes, irrespective of the content of the questions. Properly constructed questionnaires generally control for this response set by counterbalancing (2). Also called acquiescence response style . See also balanced SCALE, counterbalancing (2), RESPONSE SET (2).
