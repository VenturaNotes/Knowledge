## Source [^1]
- A benchmark program built up from a carefully selected mix of computer instructions and data types selected to be 'typical' for scientific calculations. The whetstone metric so obtained has been widely used to measure comparative processing performance of hardware/software systems. The metric was developed by ICL at Whetstone, England.
