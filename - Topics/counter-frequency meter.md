## Source [^1]
- An instrument that can be used as a counter or frequency meter by counting the number of events or periods occurring in a given time. It contains a frequency standard, usually a piezoelectric oscillator. The time between events may also be counted by comparing the number of standard pulses occurring in the same time as a given number of cycles of the frequency standard.
