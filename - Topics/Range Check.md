## Source [^1]
- A type of [[Data validation]]
- A range check will verify whether input data falls within a predefined range. 
- Example
	- Latitude and longitude are commonly used in geographic data. A latitude value should be between -90 and 90, while a longitude value must be between -180 and 180. 
		- Any values out of this range are invalid.
