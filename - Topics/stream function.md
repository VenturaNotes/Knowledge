## Source [^1]
- For an inviscid, incompressible, planar flow, with flow velocity $\bf u$, a stream function is a scalar function $\uppsi$ such that curl$(\uppsi \bf{k}) = \bf u$. The stream function is constant along streamlines. See complex potential, velocity potential.
