## Source [^1]
- Given a linear map $T:V \to V$ of an $n$-dimensional vector space, a vector $v$ is cyclic if $v,Tv,T^2v,\ldots,T^{n-1}v$ is a basis for $V$. The matrix for $T$ with respect to this basis is the companion matrix of the minimal polynomial of $T$.
