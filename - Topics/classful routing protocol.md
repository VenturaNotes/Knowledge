## Synthesis
- A type of [[routing protocol]] that uses a 32-bit IP address to divide a network into a network ID and a host ID
- Used in the early days of the internet and are still used in some legacy networks
## Source[^1]
- IGRP is a classful routing protocol
	- This is because it does not support variable length subnet masking (VLSM)
- Classful routing protocols divide the [[IP addresses|IP address]]
