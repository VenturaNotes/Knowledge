## Source [^1]
- $n$. The learning theory of the US psychologist Edwin R. Guthrie (1886-1959), according to which an organism invariably gives the same response to a stimulus (1) pattern that it gave when it last encountered the pattern (including internal stimuli), the function of reinforcement being to create a new stimulus pattern to which the organism responds similarly in the future.
