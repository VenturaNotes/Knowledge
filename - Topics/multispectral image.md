## Source [^1]
- An image produced by combining many images of the same object, each taken in a different narrow wavelength band. The selection of which wavelength bands to use is based on the kind of analysis that is planned for the resulting image. Often, when the chosen wavelengths are not those of visible light, they can be rendered using visible colors to give a false-color image. See also HYPERSPECTRAL IMAGE.
