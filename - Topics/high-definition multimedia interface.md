## Source [^1]
- (HDMI) A standard developed for physical connectors, signals, and protocols for video and multichannel audio that support high-definition resolution. Support initially was for resolutions up to WUXGA, but now extends to 4K HD. The interface also supports the bidirectional transmission of control information, so that a single remote control can be used to manage many HDMI-connected devices.
