## Source [^1]
- (complex analysis) A point $a$ is not a branch point of a multifunction $f$ if there is a holomorphic branch of $f$ on some punctured disc $0<|z-a|<\varepsilon$ around $a$ and otherwise $a$ is a branch point of $f$. Thus, 0 is a branch point of $\sqrt{z}$ and also of $\log z$.
