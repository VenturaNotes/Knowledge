## Source [^1]
- A term used to describe a network in which the total delay experienced by data traversing the network can be guaranteed not to exceed some predetermined value. This is especially important where the network is used to carry speech, as the human ear is very intolerant of breaks in speech. See also NETWORK DELAY.
