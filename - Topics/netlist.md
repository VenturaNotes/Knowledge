## Source [^1]
- A textual file listing the descriptions of objects and their interconnections required to generate a specific function or design. These objects could represent an ASIC design in the form of a set of library cells or an electronic circuit design as a list of components to be analysed by a simulation package, for example SPICE (see TRANSISTOR PARAMETERS) or PSPICE. See DESIGN AUTOMATION.
