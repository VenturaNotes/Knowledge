## Source [^1]
- An area of memory used to store information related to the pixels of a display. A frame buffer for a color display has a set of planes each defining one bit of the color information of each point. Typically a color display will have a frame buffer with up to 24 planes, 8 each for defining red, green, and blue values (see RGB COLOR MODEL).
