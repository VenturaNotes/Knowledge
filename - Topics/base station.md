## Synthesis
- A fixed radio transmitter and receiver that acts as the central hub for wireless communication within a network
## Source[^1]
- A base station in a mobile radio system is a radio receiver that communicates with mobile stations, which connect cellular calls to a public telephone network. It can also communicate directly with other base stations and the public telephone network. Each base station covers a geographical area called a cell and belongs to a specific telecommunications company. However, rival companies can have their own base stations in the same geographical area.
## Source[^2]
- A fixed radio communications station that can communicate with mobile radio devices in its neighborhood. Forms of radio communication that have base stations include radio wireless data and mobile telephony; in the latter case, the base station also connects to fixed landline data or telephone networks and routing facilities. See also CELL TOWER.
