## Source [^1]
- (Gauss error function) The error function erf(x) is a special function defined as $$erf(x) = \frac{2}{\sqrt \pi}\int_0^xe^{-t^2}dt.$$If $\Phi(x)$ denotes the cumulative distribution function of the normal distribution $N(0, 1),$ then $$erf(x) = 2\Phi(x\sqrt 2)-1$$
