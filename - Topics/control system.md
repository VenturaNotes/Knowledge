## Source [^1]
- A system that can be classified as either a process control system or a servomechanism. In a process control system the controlled variable, or output, must be held as close as possible to a usually constant desired value, or input, despite any disturbances. With a servomechanism the input varies and the output must be made to follow it as closely as possible.
