## Source [^1]
- A square matrix in which every element lying to one side of the main diagonal is equal to zero. Thus for a lower triangular matrix, $L$,$$l_{i j}=0 \text { if } i<j$$and for an upper triangular matrix, $U$,$$u_{i j}=0 \text { if } i>j$$if, in addition,$$l_{i i}=0 \text { or } u_{i i}=0$$then $L$ or $U$ is said to be strictly lower or strictly upper triangular respectively. The inverse of a lower (or an upper) triangular matrix, if it exists, is easy to calculate and is itself lower (or upper) triangular.
## Source[^2]
- A square matrix that is either lower triangular or upper triangular. It is lower triangular if all the entries above the main diagonal are zero, and upper triangular if all the entries below the main diagonal are zero. A triangular matrix is strictly triangular if the diagonal entries are also zero.
