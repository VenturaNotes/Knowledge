## Source [^1]
- A switched service network (SSN) is a term that is used within the telecommunications industry to refer to a telephone network that allows connections to be switched between users within the same network. This term can also refer to the traditional dial-up telephone system, which allows users to connect with anyone in the world by simply dialing the number.
