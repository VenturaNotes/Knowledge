## Source [^1]
- A rearrangement (or permutation) in which no object returns to its original place. By the inclusion-exclusion principle, there are $$n!(1 - \frac{1}{2!}+\frac{1}{3!}-...+\frac{(-1)^{n+1}}{n!})$$derangements of a set of elements.
