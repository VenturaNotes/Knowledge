## Source [^1]
- A process in which flash memory devices suffer incremental damage with repeated data write/erase operations (see PROGRAM-ERASE CYCLE). This is as a result of memory processes that involve high voltages and hot electron injection. Its effects can be mitigated by wear leveling.
