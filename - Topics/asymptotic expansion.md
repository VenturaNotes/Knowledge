## Source [^1]
- A series$$A_{0}+A_{1} / z+A_{2} / z^{2}+\ldots+A_{n} / z^{n}+\ldots$$that may either converge for large values of $|z|$ or diverge for all values of $z$, is called an asymptotic expansion of the function $F(z)$, valid in a given range of values of $\arg z$, if, for every fixed value of $n$, the expression$$z^{n}\left(F(z)-A_{0}-A_{1} / z-A_{2} / z^{2}-\ldots-A_{n} / z^{n}\right)$$tends to zero as $|z| \rightarrow \infty$, while $\arg z$ remains in the given range.
