## Synthesis
- A [[network access server]] that authenticates, authorizes, and accounts for users and devices trying to access a network
- RADIUS stands for Remote Authentication Dial-In User Service
## Source[^1]
- It is a networking protocol that authorizes and authenticates users who access a remote network
