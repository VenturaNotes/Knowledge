## Source [^1]
- Any device that handles remote logins to establish a point-to-point protocol connection
- Some people call these devices media access gateways or remote access servers
	- These tools handle authentication and ensure that people can tap into the resources they need
	- Might be used between users and a phone system or users and the internet
