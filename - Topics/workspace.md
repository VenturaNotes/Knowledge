## Source [^1]
- Work area is another name for workspace
- A block of locations within the main memory that are used for the temporary storage of data during processing. The user of an application such as a spreadsheet, word processor, or statistical package perceives the workspace as the space containing, and hence limiting, the tables, graphics, documents, data matrices, etc., that are currently being operated upon.
