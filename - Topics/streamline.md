## Source [^1]
- For a fluid with flow velocity $\bf u$, a streamline is a curve such that at every point of the curve a tangent vector is parallel to the flow velocity. In a steady flow, the streamlines are the paths followed by particles, but this is not generally true in an unsteady flow. See stream function.
