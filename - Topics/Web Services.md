## Source [^1]
- Web services allow communication between various network applications. They use standardized web protocols such as Simple Object Access Protocol (SOAP) or Hypertext Transport Protocol (HTTP). This allows users on various operating systems to use the same software and applications. This promotes the creation and use of web-based applications that work within internet browsers.
