## Source [^1]
- (HDL) A computer-based language that allows designers to model their hardware designs in software and simulate them, before building the actual hardware. This approach has been driven by the increasing complexities in hardware. HDL allows designs to be tested, in software, and design faults to be identified and corrected before the final expensive hardware is committed. See also VHDL.
