- If $\left\{\left(x_{i}, f_{i}\right)\right\}, i=0,1,2, \ldots$ is a given set of function values with $x_{i+1}=x_{i}$ $+h, f_{i}=f\left(x_{i}\right)$ then the backward difference at $f_{i}$ is defined by $f_{i}-f_{i-1}=f\left(x_{i}\right)-f\left(x_{i-1}\right)$. Compare FORWARD DIFFERENCE.
