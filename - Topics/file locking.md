## Source [^1]
- A method of ensuring that if one process is altering the contents of a file, other processes cannot access the file until the updating activity has been successfully completed. The actual operation of the file-locking mechanism will normally form a part of the operating-system services that allow access to files, and the lock is applied to the file in its entirety. See also RECORD LOCKING, SEMAPHORE.
