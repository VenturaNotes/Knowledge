## Source [^1]
- A (generally) binary variable describing the state of each memory element within a sequential circuit.
## Source[^2]
- A named component of a larger state space. A common way to structure complex states is to view them as composed of a collection of named sub-parts, each of which can take on some suitable range of values. A given system state is then characterised by specific values associated with each state variable. The size of a (discrete) state space composed of variables is equal to the product of the number of values associated with each state variable.
