## Source [^1]
- One of a set of buttons in a dialogue box that, when activated, allows only one of a set of mutually exclusive options to be selected at a time (as when selecting a radio station using a button with a pre-set frequency). For example, there may be a set of radio buttons with the options to print a full document, the current page, or multiple pages. Compare CHECK BOX.
