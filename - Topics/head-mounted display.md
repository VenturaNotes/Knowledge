## Source [^1]
- A headset with two stereoscopic images that give the person wearing the device a three-dimensional view of a virtual scene.
## Source[^2]
- (HMD) A display mounted in a helmet or goggles, so that it is directly in front of the user's eyes. Some systems are fully immersive in that the user can see only what is projected onto the display, whereas see-through versions mix display images with a real-world view. Applications include gaming, engineering, aviation, and military tactical information. See also AUGMENTED REALITY; VIRTUAL REALITY SYSTEMS.
