## Source [^1]
- A binary relation $\sim$ on a set S is antisymmetric if, for all $a$ and $b$ in S, whenever $a \sim b$ and $b \sim a$, then $a = b$. For example, the relation $\le$ on the set of integers is antisymmetric, as is any partial order. Compare ASYMMETRIC RELATION
