## Source [^1]
- A complex-valued function on an open subset $S$ of the complex plane that is holomorphic on all of $S$ except for a set of isolated points which are poles of the function. Equivalently a meromorphic function can be considered as a holomorphic function from $S$ to the Riemann sphere.
