## Source [^1]
- For a graph G, the maximum number of colors needed so that all regions touching one another (meeting at an edge or a vertex) are in a different color is the chromatic number, denoted by $\chi (G)$. The Four Color Theorem proved that $\chi(G) \le 4$ for all planar graphs.
