## Source [^1]
- A type of automatic frequency control in a radio receiver that adjusts the tuning to the correct setting for a given received signal when the manually operated adjustment has been set only approximately for that signal. It holds the tuning at the correct setting despite any small drift in the components of the receiver and any small drift in the frequency of the received signal.
