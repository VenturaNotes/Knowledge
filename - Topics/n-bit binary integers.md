### Definition
- The binary number system is a base 2 system consisting of only two digits, 1 and 0

### Addition
- ![[Screenshot 2023-08-08 at 6.28.28 AM.png|300]]
	- When adding "0111" + "1110", there are 4 possible results
		- 0 + 0 = 0 meaning you bring down a 0
		- 0 + 1 = 1 meaning you bring down a 1
		- 1 + 1 = 2 meaning you bring down a 0 and carry a 1 to a higher place value
		- 1 + 1 + 1 (carried over) = 3 meaning you bring down a 1 and carry over a 1.

### Examples
- A 1-bit binary integer can represent 2 values: 0 and 1
- A 2-bit binary integer can represent 4 values: 00, 01, 10, and 11
- A 3-bit binary integer can represent 8 values: 000, 001, 010, 011, 100, 101, 110, 111
