## Source [^1]
- The space where interactions between humans and machines occur, aiming for effective operation and control of the machine from the human end and feedback from the machine to the user.
## Source[^2]
- (1) The means of communication between a human user and a computer system, referring in particular to the use of input/output devices with supporting software. Examples include the graphical user interface (GUI) and command-line interface (CLI). 
- (2) Another name for front end.
