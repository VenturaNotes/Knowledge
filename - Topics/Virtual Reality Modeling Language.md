## Source [^1]
- Virtual Reality Modeling Language, or VRML, is a file type that is used to create 3D interactive vector graphics. The language uses the WWW format and is described as “worlds” with file extensions titled “.wrl.” The language is comparable to the more recognized HTML; however, HTML documents are two-dimensional instead of three.
