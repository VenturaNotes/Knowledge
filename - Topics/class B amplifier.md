## Source [^1]
- A linear amplifier operated so that the output current is cut off at zero input signal, i.e. the angle of flow equals π, and a half-wave rectified output is produced. Two transistors are required in order to duplicate the input waveform successfully, each one conducting for half of the input cycle (see PUSH-PULL OPERATION). Class B amplifiers are highly efficient but suffer from crossover distortion.
