## Source [^1]
- (MIMO) 
- (1) A control system based on multiple variables with many inputs and outputs. 
- (2) A radio system that uses multiple antennae spread over different locations for both transmitting and receiving. This enables the system to compensate for transmission errors caused by fading and multipath effects, by sending the same data over different paths at different frequencies.
