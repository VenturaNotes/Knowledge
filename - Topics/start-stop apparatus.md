## Source [^1]
- A device used in telegraphy in which each set of coded signal elements corresponding to a transmitted character is preceded by a start signal that activates the receiving apparatus in preparation for the character and is followed by a stop signal that brings the receiving apparatus to rest. Start-stop apparatus is used in telex systems. See also TELEPRINTER.
