## Source [^1]
- (1) In a combinational circuit, the maximum number of logic gates between any input and any output. The logic level represents a measure of delay time through such a circuit. 
- (2) Either of the two voltage levels used in a binary logic gate. See also MULTIVALUED LOGIC.
