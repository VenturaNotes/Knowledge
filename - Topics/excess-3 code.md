## Source [^1]
- An 8421 code for which the weighted sum of the four bits in each codeword is three greater than the decimal digit represented by that codeword. For example, 9 is represented by 1100 , the weighted sum of which is$$8 \times 1+4 \times 1+2 \times 0+1 \times 0=12$$
