## Source [^1]
- The values $Z_{i1}$ and $Z_{i2}$ of impedance that simultaneously satisfy the following conditions: if $Z_{i1}$ is connected across one pair of terminals of a two-port network then the impedance between the second pair is $Z_{i2}$; if $Z_{i2}$ is connected across the second pair of terminals then the impedance across the first pair is $Z_{i1}$.
