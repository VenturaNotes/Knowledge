## Source [^1]
- A computer program that makes decisions or solves problems by applying a set of 'if-then' rules. There are numerous applications: for example a program can be used to determine the kinds of special offers, promotions, and information that should be provided for visitors to a company's website. An expert system is a sophisticated form of rule-based system.
