## Source [^1]
- (FIFO) An accounting convention which assumes that when a firm uses materials, it uses first those which it has held in stock longest, so that the first stocks to enter its stores are the first to leave. FIFO is is the opposite convention to last-in, first-out (LIFO), under which the first stocks to be used are those most recently acquired.
