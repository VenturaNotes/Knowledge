## Source [^1]
- When a computer program is used to solve a linear programming problem, the solution can provide further information, e.g. about any slack there is in the variables, or the value attached to relaxing a constraint. Since these pieces of information are available only after the algorithm has produced the optimal solution, the general term ‘post-optimal analysis’ is used to describe procedures to obtain them.
