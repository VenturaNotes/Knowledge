## Source [^1]
- (of an array) The array obtained by constraining the subscripts to lie in some specified subrange. For example, the trim of a vector$$v=\left(v_{1} v_{2} \ldots v_{10}\right)$$obtained by constraining the index $i$ so that $3 \leq i \leq 7$ is the vector$$\left(v_{3} v_{4} v_{5} v_{6} v_{7}\right)$$See also SLICE.
