## Source[^1]
- A utility program that allows the user to locate and retrieve information from networked information services. The term is now most frequently used as a synonym for Web browser.
## Source[^2]
- A browser (or internet browser) is a type of software that accesses the World Wide Web. It displays information from web pages on a computer or mobile device. Most browsers display text, images and graphics from the internet. However, some browsers might require additional "plug-ins" to process multimedia such as audio and video. Browsers connect webpages via hyperlinks classified with uniform resource identifiers (URIs).
