## Source [^1]
- (filter synthesis) A systematic technique of determining the network transfer function that will produce a given output signal for a given input signal. This is in contrast to the normal analysis of a given network to determine the output signal given the input signal. The answer provided by the process of network synthesis is never unique, and with some combinations of input signal and required output signal an answer may not be possible.
