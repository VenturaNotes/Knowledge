## Source [^1]
- An imaging sensor built using MOSFET complementary transistors. CMOS sensors are very energy-efficient and fast, with each pixel sensor linked to its own amplifier. They are more prone to noise, but recent techniques to mitigate this have been developed; in one method a photo is taken with the shutter closed to measure the noise, which is then subtracted from the photo taken with the shutter open. See also CCD SENSOR.
