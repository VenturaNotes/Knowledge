## Source [^1]
```python
# Given
a = 50
b = 10

# If statement greater than
if a > b: 
	print("Hello World")

# If statement not equal to
if a != b:
	print("Hello World")

# If/elif/else sequence
if a == b:
	print("1")
elif a > b:
	print("2")
else:
	print("3")

# Checking two statements true
if a == b and c == d: #use "or" for at least one statement to be true
	print("TRUE")

# Short hand syntax for if else statement
print("YES") if a == b else print("NO")

```
