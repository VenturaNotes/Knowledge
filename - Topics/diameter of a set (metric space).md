## Source [^1]
- A bounded metric space X with metric $d$ has diameter $D$ if $\mu(x, y) \le D$ and $D$ is the smallest value for which the inequality holds for all pairs $x, y$ in $X$. So $D$ is the least upper bound of distances between points in the metric space
