## Source [^1]
- (1) One of the basic actions performed on a set on whose elements a total ordering $\leq$ is defined; when applied in the form $\min (S)$ it produces the smallest element of the set $S$ with respect to $\leq$. 
- (2) A monadic operation applied to a language $L$ and defined in such a way that $\min (L)$ is the set of strings in $L$ that have no proper prefixes that are also in $L$.
