## Source [^1]
- A circuit with two or more input terminals that produces an output signal when a signal is received by only one input; there is no output when a signal is received by each input either simultaneously or within a specified time interval, $\Delta t$. An electronic counter incorporating such a circuit will record single events and is termed an anticoincidence counter. See also COINCIDENCE CIRCUIT.
