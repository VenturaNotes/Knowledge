## Synthesis
- An end-user is the person who uses an [[Applications|application]] or [[services|service]] directly, such as someone browsing the web or checking email.
	- #comment An application is a software program. In this case, the end-user would be using a web browser (an application) to browse the web
- A person who interacts with the [[Applications|application]]
