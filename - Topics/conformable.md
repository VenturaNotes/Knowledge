## Source [^1]
- Matrices $\mathbf{A}$ and $\mathbf{B}$ are conformable (for multiplication) if the number of columns of $\mathbf{A}$ equals the number of rows of $\mathbf{B}$. Then $\mathbf{A}$ has order $m\times n$ and $\mathbf{B}$ has order $n \times p$, for some $m, n, \text{ and } p$, and the product $\mathbf{AB}$, of order $m\times p$, is defined. See multiplication (of matrices).
