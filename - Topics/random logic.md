## Source [^1]
- A term generally used to describe a relatively nonuniform digital logic circuit. For example, a control unit will contain random logic; an ALU, having a regular structure, will not.
## Source[^2]
- A logic circuit that contains an arrangement of different interconnected logic gates rather than linked arrays of similar gates.
