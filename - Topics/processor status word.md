## Source [^1]
- (PSW) A word that describes fully the condition of a processor at each instant. It indicates which classes of operations are allowed and which are forbidden, and the status of all interrupts associated with the processor. It will also contain the address of the instruction currently being executed. The PSW is held in a register known as the processor status register. See also PROGRAM STATUS WORD.
