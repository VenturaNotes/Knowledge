## Source [^1]
- The condition that arises when the result of an arithmetic operation exceeds the size of the location allotted for receipt of that result, or the amount by which the result exceeds the allotted number.
## Source[^2]
- The state in which a number is too large for the intended representation (for example, any value larger than 255 for an 8-bit binary representation).
