## Source [^1]
- A system for rapid development of Microsoft Windows applications. The user designs a visual interface by selecting 'controls' (e.g. menus, dialogue boxes) from a predefined palette, and then provides code in a dialect of Basic to specify the actions for each control in response to user input. The latest version is Visual Basic 2005 (see Visual BASIC .NET).
