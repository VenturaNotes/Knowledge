## Source [^1]
- Each of the hyperbolic functions sinh and tanh is strictly increasing throughout the whole of its domain $\mathbb{R}$, so in each case an inverse function exists. In the case of cosh, the function has to be restricted to a suitable domain (see INVERSE FUNCTION), taken to be $[0, \infty]$. The domain of the inverse function is, in each case, the image of the original function (after the restriction of the domain, in the case of cosh). The inverse functions obtained are: $cosh^{-1}:[1, \infty) \to [0, \infty); sinh^{-1}:\mathbb{R} \to \mathbb{R}; tanh^{-1}:(-1, 1) \to \mathbb{R}.$ These functions are given by the formula$$ \begin{align*}  cosh^{-1}x &= ln(x + \sqrt{x^2-1}) \text{ for } x \ge 1, \\
  sinh^{-1}x&=ln(x + \sqrt{x^2+1}) \text{ for all } x, \\
  tanh^{-1}x &= ln\sqrt{\frac{1+x}{1-x}} \text{ for } -1 < x < 1.\end{align*}  $$The following derivatives can be obtained $$\begin{align*}  \frac{d}{dx}(cosh^{-1}x) &= \frac{1}{\sqrt{x^2-1}} (x\ne 1), \\ \frac{d}{dx}(sinh^{-1}x)&=\frac{1}{\sqrt{x^2+1}} \\ \frac{d}{dx}(tanh^{-1}x) &= \frac{1}{1-x^2}. \end{align*}$$
