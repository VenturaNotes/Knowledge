## Source [^1]
- A variable in an equation or expression whose possible values have been restricted in some way. For example, in the equation $a + b = c$, we can consider all three variables to be unbound, or free, in that they can take on arbitrary numeric values. Some of these values will result in the equation being true, while others will make it false. However, if we qualify the equation, for example, asking for the $b$ that satisfies $a + b = c$, then $b$ becomes bound and is now a dummy variable, whose value depends on those of $a$ and $c$, which are still free. Dummy variables can always be replaced systematically by another variable not already in use without altering the meaning of the equation. In an integral such as $\int f(x)dx$, $x$ is a dummy variable, so we do not change the meaning by rewriting it as $\int f(t)dt$.
