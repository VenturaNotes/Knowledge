## Source [^1]
- (characteristic function) The indictor function of a subset $S \subseteq X$ is the function $1_S: X \to \mathbb{R}$  such that $1_S(x) = 1$ if $x \in S$ and $1_S(x) = 0$ otherwise. Such functions can be useful in probability as random variables indicating whether a certain event occurred.
