## Source [^1]
- For an equivalence relation $\sim$ on a set S, the equivalence class $[a]$, or $\overline a$, of $a \in S$ is the set of elements of S equivalent to $a$; that is, $[a] = \{x | x \in S \text{ and }a \sim x\}$. The collection of distinct equivalence classes partition $S$.
