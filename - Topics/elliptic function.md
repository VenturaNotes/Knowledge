## Source [^1]
- A meromorphic doubly-periodic function defined on the complex plane such that $f(z+ma+nb) = f(z)$ for all integers $m, n,$ where $a/b$ is not real. Note that such an elliptic function defines a meromorphic function on the elliptic curve $\mathbb{C}/\Lambda$, where $\Lambda$ denotes the lattice $a \mathbb{Z} + b\mathbb{Z}$.
- Note further by Liouville's theorem that holomorphic elliptic functions are constant. Elliptic functions became of interest as a consequence of elliptic integrals.
## Source[^2]
- The function that describes the magnitude response $H(\omega)$ of an elliptic filter and is given by:$$H(\omega) = \frac{1}{\sqrt{1 + \varepsilon R_n^2(\omega)}}$$where $\varepsilon$ is a constant that determines the amount of ripple in the pass-band and stop-band, and$$R_n(\omega) = \frac{[\omega (\omega_1^2 - \omega^2)(\omega_2^2 - \omega^2)\dots(\omega_k^2 - \omega^2)]}{[(1 - \omega_1^2 \omega^2)(1 - \omega_2^2 \omega^2)\dots(1 - \omega_k^2 \omega^2)]}$$
