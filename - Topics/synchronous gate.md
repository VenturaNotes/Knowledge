## Source [^1]
- (clocked gate) A gate the output of which is synchronized to the input signal. The synchronizing signals may be derived from an independent clock so that the gate is operative during predetermined intervals of time; alternatively the input signal itself may be used as the trigger so that the gate operates only when an input signal is present.
