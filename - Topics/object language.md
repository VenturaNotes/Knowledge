## Source [^1]
- The language in which the output of a compiler or assembler is expressed.
## Source[^2]
- $n$. A language being referred to in a metalanguage. The sentence 'Snow is white' if and only if snow is white is expressed in a metalanguage, and the sentence that it contains, enclosed in quotation marks, belongs to the object language. Such an object language may be a metalanguage relative to a third language.
