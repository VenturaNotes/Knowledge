## Source [^1]
- $n$. Loss of memory for events or experiences after the traumatic event or incident that caused the amnesia. It generally affects declarative memory but not procedural memory. See also global AMNESIA, TRANSIENT GLOBAL AMNESIA. Compare RETROGRADE AMNESIA. \[From Latin anterior previous + gradus a step]
