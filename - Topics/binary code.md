## Source [^1]
- A code whose alphabet is restricted to $\{0,1\}$. In general, any q-ary code has the important special case $q=2$. See BINARY SYSTEM.
## Source[^2]
- A code using the alphabet {0, 1}.
## Source[^3]
- $n$. A system of representing information by using the digits 0 and 1 , so that each number, letter, or other item of information is represented by a unique sequence of bits.
