## Source [^1]
### Purpose
- Prepares source code for compiler proper
- Example in C/C++
	- Consumes all directives starting with # symbol
		- The \#include directive causes preprocessor to open named file and insert its contents into source code
		- A \#define directive causes preprocessor to substitute a value when a macro name is encountered
