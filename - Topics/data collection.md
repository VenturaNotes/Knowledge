## Source [^1]
- The process of collecting data from distributed points at which it has been captured or input as a separate operation. Generally the equipment used for the process is connected to a host computer via a communication system; sometimes portable equipment is carried to each site for information to be input into its memory and then the equipment is connected to the host system. See also DATA CAPTURE.
