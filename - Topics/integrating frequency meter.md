## Source [^1]
- (master frequency meter) An instrument that allows a check to be made on a source of alternating voltage by integrating the number of cycles that occur in a specified time interval. Comparison with the number of cycles that should occur in the same time interval indicates whether the prescribed frequency has been maintained.
