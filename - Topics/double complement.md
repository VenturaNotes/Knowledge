## Source [^1]
- (of a set $S$ ) The complement of the set $S^{\prime}$, where $S^{\prime}$ itself is just the complement of $S$; the double complement of $S$ is thus $S$ itself. In logic double complement implies double negation of an element $x$, say, i.e. $x$ itself.
