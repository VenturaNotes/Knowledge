## Source [^1]
- An integral with explicit integration limits that serves to fix the arbitrary constant associated with the solution, so that the integral denotes a single value. Given an indefinite integral$$\int f(x) \, dx$$with solution $F(x) + C$, with $C$ arbitrary, then the definite integral is defined as$$\int_{a}^{b} f(x) \, dx = F(b) - F(a)$$
