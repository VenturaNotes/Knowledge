## Source [^1]
- A ratio that measures the efficiency of labour or an activity over a period by dividing the standard hours allowed for the production by the actual hours taken. It is usually expressed as a percentage, using the formula:$$\begin{gathered}\text { (standard hours allowed } \times 100) / \\\text { actual hours worked. }\end{gathered}$$
