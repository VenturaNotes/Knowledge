## Source [^1]
- Digital logic restricted to the description of combinational circuits. See also BOOLEAN EXPRESSION
## Source[^2]
- A logic circuit whose output at a specified time is a function only of the inputs at that time and does not depend on the history of inputs to the circuit. Hence it is a circuit without any memory components or any feedback loops. Compare SEQUENTIAL CIRCUIT.
