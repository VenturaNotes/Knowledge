## Source [^1]
- A peer-to-peer network formed by a set of nodes that dynamically configure themselves in the form of a temporary network. An ad hoc network is typically a wireless LAN configuration. A MANET (Mobile Ad hoc NETwork) is an ad hoc network in which all the nodes are wireless mobile nodes capable of self-organizing themselves dynamically. See also WIRELESS SENSOR NETWORK.
