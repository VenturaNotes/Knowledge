## Source [^1]
- An object-oriented programming concept that binds together the data and functions that manipulate the data and that keeps both safe from outside interference and misuse.
## Source[^2]
- Encapsulation refers to how data is stored and accessed within an object or class in programming languages. It helps protect sensitive information from being leaked outside of its intended scope (i.e., private) while also enabling access for other objects that need it (public).
