### Description
- The [[del (Python)|del]] keyword is used to delete objects. In Python, everything is an object, so the del keyword can also be used to delete variables, lists, or parts of a list etc. [^1]

### Example 
Input
```python
x = ["apple", "banana", "cherry"]

del x[0]

print(x)

```

Output
```python
['banana', 'cherry']
```
