## Source [^1]
- A memory IC (integrated circuit) whose pin-out corresponds to the SIL format. Because the pins lie along one edge of the device package, it can be mounted on a PCB in a perpendicular plane, minimizing the board area occupied and maximizing the packing density. See also MEMORY CARD.
