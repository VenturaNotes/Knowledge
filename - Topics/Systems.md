## Synthesis
### Description 1
- A system is a collection or set of equations that are considered together
- The goal is to find values for the variables that satisfy all of the equations simultaneously
### Description 2
- A system over a [[network]]
	- Refers to the entire computing setup, including hardware, software, and the operating system, that communicates over the network
	- Different from [[Applications|application]]
