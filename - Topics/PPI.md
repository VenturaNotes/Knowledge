## Source [^1]
- (1) (pixels per inch) A measure of screen density, being the number of pixels on a screen contained in a distance of one inch, either horizontally or vertically. It is an indicator of the perceived sharpness of the screen. It can be determined by dividing the horizontal screen resolution by the width of the screen in inches. 
- (2) See PLAN POSITION INDICATOR.
