## Source [^1]
- The frequency range within which a phase-lock loop (PLL) will capture and lock on to the input frequency. If the input signal range is greater than the PLL lock range, the input signal will need to be searched for by changing the frequency of the voltage-controlled oscillator before lock and tracking will result.
