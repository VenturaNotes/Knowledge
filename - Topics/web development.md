## Source [^1]
- The application of software development to the creation of a website. This encompasses development of both static websites, which do not possess a database, and dynamic websites, which contain databases and may be arbitrarily complex. Web development activities include content development, web design, and security configuration. A web developer is a person responsible for the development of a website.
