## Source [^1]
- (SSHD) A computer storage device constructed from both solid-state NAND flash circuits and a magnetic disk hard drive. The flash memory has a smaller capacity than the hard disk and acts as cache memory for the disk. This results in a nonvolatile memory that is faster than one based purely on magnetic disks, but with a lower cost per bit than a pure solid-state drive. SSHDs are a common mid-range option for personal computing devices.
