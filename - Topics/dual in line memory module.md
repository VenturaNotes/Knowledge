## Source [^1]
- A printed circuit board carrying RAM chips which at present (2008) can provide up to 1 Gbyte of memory on a module approx. 135 $\mathrm{mm} \times 25 \mathrm{~mm}$. DIMMs are typically plugged into edge connectors on a motherboard. Bus speeds up to 667 MHz are common.
