## Source [^1]
- Denoting a curve or surface divided into a number of pieces that are continuous to a specified degree across the joins.
## Source[^2]
- A function is piecewise continuous on a bounded interval if the interval can be subdivided into a finite number of subintervals on which the function is continuous (see CONTINUOUS FUNCTION) with finite limits at each subinterval’s endpoints.
