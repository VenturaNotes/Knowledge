## Source [^1]
- A complex-valued function $\psi$ which describes the state of a system in quantum theory and which satisfies  Schrödinger's equation. The wave function is normalized if it has unit norm $||\psi||$, in which case $|\psi|^2$ is a probability density function for the particle's position.
