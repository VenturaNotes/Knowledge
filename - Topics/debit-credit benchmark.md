## Source [^1]
- A benchmark test for measuring the number of office tasks completed on a computer installation per hour. The basic feature of the test is to use a fixed number of terminals on a given computer configuration and to ramp up the number of standardized transactions per second until the system can only complete $95 \%$ of requested transactions in under one second.
