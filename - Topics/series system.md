## Source [^1]
- A system in which devices with a single input and a single output are connected in a chain, so that the output of each device is connected to the input of the next (see SERIES). Devices can range from simple circuit elements to more complex signal processing or data-transformation elements. Compare PARALLEL SYSTEM.
