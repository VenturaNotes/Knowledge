## Source [^1]
- Provisioning is the process of configuring and deploying a technical resource either locally or through the cloud. Provisioning can also refer to the steps taken by service providers to allow users access to specific data and resources. There are various types of provisioning in computing, including server provisioning, user provisioning, network provisioning, and service provisioning. Server provisioning is the process of setting up a server to be used within a network. User provisioning monitors user access rights and authorization privileges. Network provisioning involves setting up a network to be accessed by various computerized devices. Service provisioning involves setting up a specific service and managing its related data.
