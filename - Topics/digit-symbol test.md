## Source [^1]
- $n$. A type of intelligence test item or subtest in which the respondent learns a code in which each digit is represented by a symbol$\textemdash$for example, 1 might be represented by a star, 2 by a square, 3 by a triangle, and so on$\textemdash$and then tries to substitute the correct symbols for a series of digits as quickly and accurately as possible. Also called a symbol-digit test or a coding test.
