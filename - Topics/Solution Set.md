## Source [^1]
- The set of all possible solutions to a system
	- You could just have 1 solution
	- Could be you have many solutions
	- Could be you have no solutions
## Source[^2]
- (solution space) The solution set of a set of equations is the set consisting of all the solutions. It may be referred to as the 'solution space' if it has further structure, for example that of an affine space or vector space. See also inequality.
