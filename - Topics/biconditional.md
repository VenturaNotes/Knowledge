- A biconditional is when the truth values of both sides coincide (meaning both true or both false), for this logical statement A $\leftrightarrow$ B to be true
- ![[Screenshot 2024-12-12 at 8.13.37 AM.png]][^2]
- $A \leftrightarrow B \iff (A \to B) \land (B \to A)$
	- The double sided arrow ($\iff$) means there is a [[tautology]] as the truth values of both sides coincide
