## Source [^1]
- A point in a topological space (or metric space) contained within an open subset of the space. When the real line is being considered, the real number $x$ is an interior point of the set $S$ of real numbers if there is an open interval $(x - \delta, x +\delta)$, where $\delta > 0$, included in S.
