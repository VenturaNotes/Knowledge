## Source [^1]
- An automatic computer check to ensure that the data entered is sensible and reasonable. It does not check the accuracy of data

## Source[^2]
- Description of Validation
	- Aims to make sure that data is reasonable
	- Aims to make sure that data is within acceptable boundaries
	- Aims to make sure that data is sensible
- Incorrect description of Validation
	- Aims to make sure that data is accurate

## Source[^3]
- The process of ensuring that a program operates on clean, correct, and useful data. It involves checks and routines for data integrity, correctness, and completeness in software applications
