### Definition
The set which contains all the elements of the universal set other than the prior set
## Source[^2]
### Examples
- Example #1
	- U = {1, 2, 3, 4, 5}
	- S = {1, 2, 3}
	- Absolute Complement of "S" is {4, 5}
		- Represented as $S' \text{ or } S^c \text{ or } \overline S$
