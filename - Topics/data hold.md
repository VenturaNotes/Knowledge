## Source [^1]
- (1) A circuit with a control input that enables the value of its data output to be frozen at its current value, rather than tracking the changes resulting from change in its input. 
- (2) A feature on some multimeters that freezes the display, retaining the last measured value, even when the probes are disconnected. This is useful when access is awkward and it is difficult to apply the probes and see the display at the same time. Similar features can also be found on logic analyzers.
