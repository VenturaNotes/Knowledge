## Source [^1]
- The value of an estimator calculated from a particular sample. If the estimate is a single figure, it is a point estimate; if it is an interval, such as a confidence interval, it is an interval estimate
## Source[^2]
- In econometrics, the value of an unknown model parameter obtained by applying an estimator to the data sample.
