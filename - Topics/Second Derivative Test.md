## Source [^1]
- If f is a function with continuous second derivative 
	- if c is a [[critical points|critical point]] and f(c) > 0, then c is a [[local minimum]]
	- If c is a critical point and f(c) < 0, then c is a [[local maximum]]
- Relates the concavity of f with classification of critical points
