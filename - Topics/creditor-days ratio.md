## Source [^1]
- A ratio that gives an estimate of the average number of days' credit taken by an organization before the creditors are paid. It is calculated by the formula: $$\begin{gathered}\text { (trade creditors } \times 365) / \text { annual } \\\text { purchases on credit. }\end{gathered}$$
