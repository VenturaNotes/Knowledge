## Synthesis
- Directly enables a user to accomplish a task
- A functional requirement describes a specific function or feature that the system must perform, which directly enables a user to accomplish a task 
	- My example is that the implementation of a resizer tool in Photoshop is a functional requirement.
### Examples in Software
1. System allows users to log in with a valid username and password
2. System displays a list of available products to user
3. System allows users to add items to shopping cart
4. System processes payment using credit card
5. System generates a daily sales report
6. System allows users to search for documents by keyword
## Source [^1]
- Product features or functions that developers must implement to enable users to accomplish their tasks

- Generally, functional requirements describe system behavior under specific conditions
## Source[^2]
- If you were to create a URL shortener, some required functional requirements may include
	- Generating a unique short URL which redirects the user to a long URL
	- Supporting short URL expiration where they are no longer accessible after a certain period.
