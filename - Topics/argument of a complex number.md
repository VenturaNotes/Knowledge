## Source [^1]
- (phase of a complex number) The angle $\theta$ that a complex number $x + iy$ makes with the x-axis when plotted using $x$ and $y$ as Cartesian coordinates. It can be computed using the formula $\theta = \tan^{-1}(y / x)$. See also INVERSE TRIGONOMETRIC FUNCTIONS.
