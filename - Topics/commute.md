## Source [^1]
- Let $\circ$ be a binary operation on a set $S$. The element $a$ and $b$ of $S$ commute (under the operation $\circ$) if a $\circ$ b = b $\circ$ a. For example, multiplication on the set of all real $2 \times 2$ matrices is not commutative, but if $A$ and $B$ are diagonal matrices, then $A$ and $B$ commute.
