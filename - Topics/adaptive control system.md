## Source [^1]
- A control system in which the controller has adjustable parameters and a mechanism for adjusting the parameters. The controller becomes nonlinear because of the parameter adjustment mechanism. An adaptive control system can be modelled as a system having two loops (see diagram). One loop is a normal feedback with the process and the controller. The other loop is the parameter adjustment loop, which is often slower than the normal feedback loop.
