## Source [^1]
- The set of technologically feasible input-output combinations for an individual firm or for an economy as a whole. The production function is the upper boundary of the production set and represents efficient input-output combinations. Points inside the production set are inefficient. A convex production set represents a technology with non-increasing returns.
