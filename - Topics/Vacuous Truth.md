## Synthesis
### Description
- A statement or logical implication that is technically true because its premise (antecedent) is false or cannot be satisfied
- It is vacuous (empty) because it doesn't provide meaningful information about the world (no objects to which it applies)
### Key Aspects
- Structure
	- $P \to Q$ (if P, then Q) where P is false
- No counterexample
	- True because impossible to present counterexample 
- Example 1 (Conditional)
	- "If pigs fly, then I am a millionaire". 
		- Since pigs can't fly (false premise), entire statement true regardless of bank account.
- Example 2 (Universal)
	- "All items in this empty box are gold"
		- Since no items in box, statement cannot be proven false making it vacuously true.

### Importance
- In mathematics and logic, crucial for maintaining consistent rules
- For example, the principle that the empty set is a subset of every set.
	- This simplifies formal proofs by eliminating special exceptions
