## Source [^1]
- (advance data communication control procedure) In communications, a type of protocol developed by the American National Standards Institute. It works at the bit level within a message: particular bits in the message are used to indicate which bits are the message, which bits are the sender and recipient identifiers, and which bits are concerned with error handling.
