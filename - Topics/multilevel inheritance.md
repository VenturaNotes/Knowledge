## Synthesis
- A class inherits from a class that already inherited from another class
- Like a chain
```python
class Animal:
    pass

class Mammal(Animal):  # Level 1
    pass

class Dog(Mammal):     # Level 2
    pass
```
- Summary
	- Inherits from a class that itself inherits
