## Source [^1]
- Part of a process that must be executed indivisibly. Originally it was thought that the indivisibility had to be absolute. Now it is considered that it is only necessary for the critical section to be uninterrupted by other critical sections of a particular set of processes, i.e. those among which there is mutual exclusion.
