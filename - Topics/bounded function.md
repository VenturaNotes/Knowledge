## Source [^1]
- A real function $f$, defined on a domain $S$, is bounded if there is a number $M$ such that $|f(x)|<M$ for all $x$ in $S$. More generally a function whose codomain is a metric space is bounded if its image is a bounded set. See WeIERSTRASS' ThEOREM.
