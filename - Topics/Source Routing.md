## Source [^1]
- Source routing is a technique used in local-area networks (LANs) that allows senders to specify the route that data packets take as they travel through a network. Senders can more easily troubleshoot data transmission since the path is not random like it is with traditional data packet routing.
## Source[^2]
- A technique in which a packet's route through a network is predetermined before transmission and encoded within the packet. Less robust than dynamic routing, in which the direction of each hop is determined by the intermediate nodes along the way, it can provide faster routing and load balancing and can help prevent thrashing.
