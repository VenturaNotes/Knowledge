## Source [^1]
- A web document that contains a summary of and link to another document. Web feeds are often used to notify subscribers to a service of updated content. They are usually collected by a program running on the user's machine called an aggregator, which periodically checks for new feeds. There is no standard format for web feeds. See also RSS.
