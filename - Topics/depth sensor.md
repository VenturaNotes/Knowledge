## Source [^1]
- A camera sensor that can estimate the distance to various points in the real world corresponding to regions within the image it has captured. Depth sensors are active, using infrared light to illuminate the scene. The simplest project is a grid of dots, with the distance between the dots on the image being used to estimate the distance of the corresponding surface. More sophisticated sensors measure time of flight or changes in the phase of the returning infrared light for greater accuracy. Applications include gesture recognition, robot navigation, and 3D dimension capture.
