## Source [^1]
- For a symmetric, positive definite real matrix $\mathbf{A}$, there exists an upper triangular matrix $\mathbf{R}$, with positive diagonal entries, such that $\mathbf{A} = \mathbf{R}^\top \mathbf{R}$. More generally, if $\mathbf{A}$ is Hermitian, then such $\mathbf{R}$ still exists with $\mathbf{A} = \mathbf{R}^* \mathbf{R}$ where denotes the Hermitian conjugate.
