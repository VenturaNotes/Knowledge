## Source [^1]
- An approach to the analysis of investment opportunities in which an investment project is assessed using the standard techniques for calculating option value. A real option is the right, but not the obligation, to undertake an investment, features it shares with a financial option. In this case, an investment opportunity is used in the broad sense of an activity that requires incurring a cost with the expectation of a reward in the future.
