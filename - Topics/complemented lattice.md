## Source [^1]
- A lattice in which there are identity elements 0 and 1 and in which each element $a$ has at least one complement $b$, i.e. $$a \land b = 0 \text { and } a \lor b = 1$$It will also be the case that $b$ is a complement of $a$ and that 0 and 1 are the complements of each other.
