## Source [^1]
- A file that can be read or copied, but not erased or modified. The protection is maintained by the operating system and in multiuser systems can be made to depend upon the category of the user. A file might be writeable by its owner, read-only to the owner's colleagues, and inaccessible to everyone else. See also FILE PROTECTION.
